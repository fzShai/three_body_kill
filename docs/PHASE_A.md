# Phase A 实现确认

基于《三体杀完整规则：理解与实现路线》第 5 节默认拍板，Phase A 按下列约定落地。

## 范围（Phase A）

1. 回合三阶段：摸牌（默认 3）→ 出牌 → 弃牌（上限 `max_hp - 2`）；出牌阶段不可跳过弃牌
2. 开局摸 6 张（科技池加权 摸牌）
3. 科技等级 1–6（开局 1）；【天外来客】+1 级；出牌阶段主动弃置 4 张基本牌 +1 级（每回合不限次）
4. 基本牌：杀 / 闪 / 桃 / 天外来客（含阶与科技映射、每回合出杀上限 2、闪响应）
5. 伤害公式：基础伤害 `基础+加成-减免`；真实伤害 `真实+加成`；向下取整；只扣最终伤害
6. 视野：bool；杀对暴露视野目标有文档规定的额外基础伤害
7. 濒死：濒死者可用桃自救；其他存活角色也可在濒死阶段对其使用桃救人。超时或濒死者点「结算濒死」时，若其手牌有桃则强制自救，否则死亡。暂不做整圈强制问询与角色濒死技
8. 重铸：无合法目标时可弃置该牌并从当前科技池再摸一张（公开战报不暴露摸到的牌）
9. 临时飞升（纳米工程中心 / 芯片车间 / 群星计划）：打出后成为可并存的正面状态（写入状态栏，不占装备栏）；同名不重复叠加
10. 阶梯计划：指定一名未暴露视野的其他角色；无此类目标时可重铸

## 非 Phase A（延后）

- 多数锦囊完整效果、虚境、场地
- 旧占位装备（恒星航迹/维稳系统）保留空栏；主路径为船/甲；临时飞升走状态而非装备槽

Phase A 收尾已含：【阶梯计划】暴露视野、重铸「无合法打法」校验。录入流水线见 [CARD_PIPELINE.md](CARD_PIPELINE.md)。

## 交付标准

2 人局能：科技摸牌、出杀→出闪响应或受伤、用桃回血、濒死自救或他人救人、天外来客升科技、主动弃置 4 基本牌升科技、弃牌阶段自选弃牌、多临时飞升状态并存、阶梯无有效目标时可重铸。
